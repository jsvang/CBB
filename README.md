# Undefeated CBB

College basketball roster builder inspired by [82-0.com](https://82-0.com). Spin team + era combos, draft a starting five, and chase an 82-0 season.

## Run locally

```bash
python -m http.server 8080
```

Open http://localhost:8080

## Rebuild player database

```bash
python scripts/build_cbb_db.py
```

## Stack

Vanilla HTML / CSS / JS — no build step. Static deploy to Cloudflare Pages or GitHub Pages.
